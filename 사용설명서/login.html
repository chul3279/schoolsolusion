로그인 및 회원가입

SchoolUs 서비스를 이용하려면 먼저 회원가입 후 로그인해야 합니다.

로그인

  • SchoolUs 홈페이지(schoolwithus.co.kr)에 접속합니다.
  • 상단 메뉴의 로그인 버튼을 클릭합니다.
  • 아이디와 비밀번호를 입력합니다.
  • 로그인 버튼을 클릭합니다.
로그인 후 역할에 따라 자동으로 해당 대시보드로 이동합니다.
- 교사: 교사 대시보드
- 학생: 학생 대시보드
- 학부모: 학부모 대시보드

회원가입

1단계: 가입 유형 선택

3가지 역할 중 하나를 선택합니다.

유형대상
학생재학 중인 학생
선생님재직 중인 교사
학부모재학생의 보호자

2단계: 소속 학교 선택

  • 학교명 검색창에 학교 이름을 입력합니다.
  • 검색 결과에서 소속 학교를 클릭하여 선택합니다.
학교명은 일부만 입력해도 검색됩니다. 예: "성문" 입력 시 "성문고등학교"가 표시됩니다.

3단계: 기본 정보 입력

항목설명
이름실명을 입력합니다
아이디사용할 아이디 입력 후 중복확인 필수
비밀번호4자리 이상
생년월일날짜 형식으로 선택
연락처010-0000-0000 형식
주소시/구/동까지 입력
아이디 중복확인을 하지 않으면 가입이 완료되지 않습니다. 반드시 중복확인 버튼을 클릭하세요.

4단계: 학부모 추가 정보 (학부모만)

학부모로 가입하는 경우 자녀 정보를 추가로 입력합니다.

  • 자녀 학교명
  • 자녀 이름
  • 자녀 학번 (예: 20301 = 2학년 3반 1번)

5단계: 가입 완료

가입 완료 버튼을 클릭하면 회원가입이 완료됩니다. 이후 로그인 페이지에서 바로 로그인할 수 있습니다.
