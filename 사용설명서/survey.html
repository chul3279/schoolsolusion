설문조사

학교에서 진행하는 설문조사에 참여하는 페이지입니다. 교사가 생성한 설문을 학생과 학부모가 응답합니다.

설문 목록

현재 진행 중인 설문 목록이 카드 형태로 표시됩니다.

  • 미응답 - 아직 응답하지 않은 설문 (보라색 배지)
  • 응답 완료 - 이미 응답한 설문 (녹색 배지, 반투명 처리)
  • 각 설문에 제목, 설명, 문항 수, 시작일이 표시됩니다.
진행 중인 설문이 없으면 "현재 진행중인 설문이 없습니다" 안내가 표시됩니다.

설문 응답 방법

  • 미응답 설문의 응답하기 버튼을 클릭합니다.
  • 각 문항에 맞는 형태로 답변을 입력합니다.
  • 모든 필수 문항에 응답합니다 (* 표시).
  • 하단의 제출하기 버튼을 클릭합니다.

문항 유형

유형설명응답 방법
객관식 (단일)보기 중 1개 선택보기를 클릭하여 선택
객관식 (복수)보기 중 여러 개 선택원하는 보기를 모두 클릭
별점1~5점 평가별 아이콘을 클릭 (매우 불만~매우 만족)
서술형자유 텍스트 입력텍스트 입력창에 직접 작성
설문은 한 번 제출하면 수정할 수 없습니다. 제출 전 답변을 꼼꼼히 확인하세요.

뒤로가기

설문 응답 중 좌측 상단 뒤로가기 버튼을 누르면 설문 목록으로 돌아갑니다. 응답 중인 내용은 저장되지 않으니 주의하세요.
