학급활동 (학부모)

자녀가 속한 학급의 정보를 확인하는 페이지입니다. 상단에 학급, 자녀 이름, 담임선생님 정보가 표시됩니다.

5개 탭 구성

하단 탭을 선택하여 각 영역을 전환합니다.

탭내용
공지학급 공지사항 조회
과제자녀의 과제 현황 확인
상담담임선생님 상담 일정 확인
출결자녀 출결 현황 조회
통신문가정통신문 확인 및 동의 제출

공지 탭

담임선생님이 작성한 학급 공지사항을 확인합니다.

  • 검색창에서 키워드로 검색할 수 있습니다.
  • 공지를 클릭하면 상세 내용이 모달로 표시됩니다.

과제 탭

자녀에게 출제된 과제 목록과 제출 상태를 확인합니다.

  • 제출완료 - 자녀가 과제를 제출한 상태
  • 진행중 - 마감 전이며 아직 미제출
  • 미제출 - 마감일이 지났으나 미제출

과제를 클릭하면 과목명, 선생님, 설명, 제출 파일 정보를 확인할 수 있습니다.

학부모는 과제를 직접 제출할 수 없습니다. 제출은 학생 계정에서만 가능합니다.

상담 탭

담임선생님이 배정한 상담 일정을 확인합니다.

  • 상담 유형: 전화 / 방문 / 온라인
  • 상태: 대기중 / 확정 / 완료 / 취소

출결 탭

자녀의 출결 현황을 요약과 상세 이력으로 확인합니다.

  • 상단: 출석, 결석, 지각, 조퇴, 병결 각각의 누계
  • 하단: 최근 출결 이력이 날짜순으로 표시됩니다.

통신문 탭

가정통신문을 확인하고 동의 여부를 제출합니다.

  • 통신문 목록에서 항목을 클릭합니다.
  • 내용과 첨부파일을 확인합니다.
  • 동의가 필요한 경우 동의 또는 미동의를 선택합니다.
  • 비고를 입력(선택)한 후 제출합니다.
동의필요 배지가 붙은 통신문은 학부모의 응답이 필요합니다. 가능한 빨리 확인하고 응답해주세요.
자녀의 학급이 배정되지 않은 경우 모든 탭의 데이터가 표시되지 않습니다.
