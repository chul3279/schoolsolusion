메시지

학교 구성원 간 1:1 또는 그룹 메시지를 주고받는 채팅 기능입니다. 메신저와 유사하지만, 각 역할(교사/학생/학부모)의 대시보드에서 직접 접근할 수 있습니다.

화면 구성

  • 대화방 목록 - 좌측에 참여 중인 대화방이 표시됩니다.
  • 채팅 화면 - 우측에 선택한 대화방의 메시지가 표시됩니다.

대화방 목록

  • 검색창에서 대화방 또는 상대방 이름을 검색합니다.
  • 읽지 않은 메시지가 있는 대화방에 빨간 배지가 표시됩니다.
  • 대화방을 클릭하면 우측에 메시지 내용이 로드됩니다.

새 대화 시작

  • 대화방 목록 상단의 + 또는 새 대화 버튼을 클릭합니다.
  • 대화 상대를 검색하고 선택합니다.
  • 1명 선택 시 1:1 대화, 여러 명 선택 시 그룹 대화가 생성됩니다.
  • 대화 시작 버튼을 클릭하면 대화방이 열립니다.

메시지 보내기

  • 하단 입력창에 메시지를 입력하고 전송합니다.
  • Enter로 전송, Shift+Enter로 줄바꿈합니다.
  • 파일 첨부 버튼으로 문서나 이미지를 함께 보낼 수 있습니다.

메시지 관리

  • 내가 보낸 메시지는 삭제할 수 있습니다.
  • 삭제된 메시지는 "삭제된 메시지입니다"로 표시됩니다.
  • 첨부 파일은 파일명 클릭으로 다운로드합니다.

날짜 구분

메시지 사이에 날짜 구분선이 자동으로 삽입됩니다. 오늘/어제는 텍스트로, 그 이전은 월/일로 표시됩니다.

대시보드의 메시지 위젯에서 새 메시지 알림을 바로 확인할 수 있습니다. 위젯을 클릭하면 메시지 페이지로 이동합니다.
모바일에서는 대화방 목록과 채팅 화면이 전체 화면으로 전환됩니다. 좌측 상단 뒤로가기로 목록에 복귀합니다.
