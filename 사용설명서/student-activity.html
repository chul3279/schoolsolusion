수업활동 (학생)

수강 중인 과목의 과제를 확인하고 제출하며, 동아리 활동 파일을 관리하는 페이지입니다.

수업 과제

과목 선택

상단에 내가 수강 중인 과목이 탭 형태로 표시됩니다. 과목을 클릭하면 해당 과목의 과제 목록이 나타납니다.

과목 목록은 선생님이 설정한 시간표를 기반으로 자동 생성됩니다.

과제 목록

각 과제에는 다음 정보가 표시됩니다.

  • 제목과 설명
  • 출제 선생님, 출제일, 마감일
  • 제출 상태: 제출완료(녹색) / 미제출(주황) / 기한초과(빨강)

과제 제출 방법

  • 과제 항목의 과제 제출 버튼을 클릭합니다.
  • 파일을 선택합니다 (PDF, HWP, DOCX, XLSX, PPTX, 이미지, ZIP / 최대 10MB).
  • 필요시 코멘트를 입력합니다.
  • 제출하기 버튼을 클릭합니다.
이미 제출한 과제는 다시 제출 버튼으로 재제출할 수 있습니다. 기존 파일이 새 파일로 대체됩니다.

동아리 활동

상단 탭에서 동아리 활동을 선택하면 소속 동아리 목록이 표시됩니다.

동아리 파일 관리

  • 동아리 카드를 클릭하여 선택합니다.
  • 파일 제출 버튼으로 활동 파일을 업로드합니다.
  • 내가 올린 파일은 내가 올림 배지가 표시되며, 삭제도 가능합니다.
  • 지도교사가 올린 파일도 목록에서 확인할 수 있습니다.
  • 파일 형식과 크기 제한은 과제 제출과 동일합니다.
동아리에 배정되지 않은 경우 동아리 목록이 비어 있습니다. 담당 선생님께 배정을 요청하세요.
