메신저

같은 학교 구성원과 실시간으로 대화할 수 있는 메신저 기능입니다. 교사, 학생, 학부모 모두 이용 가능합니다.

화면 구성

  • 좌측 - 대화 목록 패널 (대화방 검색, 새 대화 시작)
  • 우측 - 메시지 패널 (선택한 대화방의 메시지)
모바일에서는 대화 목록과 메시지 화면이 전체 화면으로 전환됩니다. 상단 뒤로가기 버튼으로 목록으로 돌아갑니다.

대화 목록

  • 최근 대화순으로 대화방이 정렬됩니다.
  • 읽지 않은 메시지가 있으면 빨간 배지로 표시됩니다.
  • 검색창에서 상대방 이름으로 대화를 검색할 수 있습니다.

새 대화 시작

  • 대화 목록 상단의 + 버튼을 클릭합니다.
  • 역할별 필터(전체/교사/학생/학부모)로 검색 범위를 좁힙니다.
  • 이름으로 검색하여 대화 상대를 선택합니다.
  • 여러 명을 선택하면 그룹 대화가 됩니다.
  • 대화 시작 버튼을 클릭합니다.

메시지 보내기

  • 하단 입력창에 메시지를 작성하고 전송 버튼(또는 Enter)을 누릅니다.
  • Shift+Enter로 줄바꿈이 가능합니다.
  • 클립 아이콘으로 파일을 첨부할 수 있습니다.

메시지 관리

  • 삭제 - 내가 보낸 메시지 하단의 삭제 버튼으로 메시지를 삭제할 수 있습니다.
  • 파일 다운로드 - 첨부 파일이 있는 메시지에서 파일명을 클릭하면 다운로드됩니다.
  • 대화방 나가기 - 우측 상단의 나가기 아이콘으로 대화방을 떠날 수 있습니다.
대화방을 나가면 해당 대화 내역을 더 이상 볼 수 없습니다.

자동 새로고침

메신저는 30초마다 새 메시지를 자동으로 확인합니다. 페이지를 벗어났다 돌아와도 최신 메시지가 자동 로드됩니다.
