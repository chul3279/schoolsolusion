공지사항

학교 전체에 공개되는 공지사항을 확인하는 페이지입니다. 교사, 학생, 학부모 모두 접근할 수 있습니다.

공지사항 목록

학교 공지사항이 번호, 제목, 작성자, 작성일 순으로 테이블에 표시됩니다.

  • 제목을 클릭하면 상세 내용 화면으로 전환됩니다.
  • 목록은 최신 순으로 정렬됩니다.

공지사항 상세 보기

선택한 공지사항의 제목, 작성자, 작성일, 본문 내용이 표시됩니다.

  • 하단의 목록으로 버튼으로 목록 화면으로 돌아갑니다.
  • 브라우저의 뒤로가기 버튼도 정상 작동합니다.

교사 전용 기능

공지사항 등록

  • 목록 화면의 글쓰기 버튼을 클릭합니다.
  • 제목과 내용을 입력합니다.
  • 수정/삭제용 비밀번호를 4자리 이상 설정합니다.
  • 등록하기 버튼을 클릭합니다.
비밀번호를 잊어버리면 수정/삭제가 불가능합니다. 안전한 곳에 메모해 두세요.

공지사항 수정

  • 본인이 작성한 공지의 상세 화면에서 수정 버튼을 클릭합니다.
  • 제목과 내용을 수정합니다.
  • 등록 시 설정한 비밀번호를 입력합니다.
  • 수정 완료 버튼을 클릭합니다.

공지사항 삭제

  • 본인이 작성한 공지의 상세 화면에서 삭제 버튼을 클릭합니다.
  • 등록 시 설정한 비밀번호를 입력합니다.
  • 삭제 버튼을 클릭하면 영구 삭제됩니다.
