회원정보 수정 (마이페이지)

개인정보를 확인하고 수정할 수 있는 페이지입니다. 모든 역할(교사, 학생, 학부모)에서 접근 가능합니다.

본인 확인

회원정보를 열람하기 전 본인 확인 절차가 필요합니다.

  • 현재 비밀번호를 입력합니다.
  • 확인 버튼을 클릭합니다.
  • 인증 성공 시 회원정보 수정 화면이 표시됩니다.

기본 정보 수정

아래 항목을 확인하고 수정할 수 있습니다.

학교 정보 변경

전학 또는 전근 시 소속 학교를 변경할 수 있습니다.

학교를 변경하면 기존 학교와 연결된 정보가 모두 업데이트됩니다. 신중하게 변경하세요.

역할별 추가 정보

교사

  • 부서, 담당업무
  • 담임 학년/반 설정

학생

  • 학년, 반, 번호 설정 (필수)

학부모

  • 자녀 이름, 자녀 생년월일 (필수)
  • 자녀 학년, 반, 번호 (필수)

비밀번호 변경

  • 페이지 하단의 비밀번호 변경하기 링크 또는 사이드바의 비밀번호 변경을 클릭합니다.
  • 현재 비밀번호를 입력합니다.
  • 새 비밀번호를 입력합니다 (8자 이상).
  • 새 비밀번호를 다시 입력하여 확인합니다.
  • 변경하기 버튼을 클릭합니다.
비밀번호 강도가 실시간으로 표시됩니다. 영문 대/소문자, 숫자, 특수문자를 조합하면 강한 비밀번호가 됩니다.

저장

수정 완료 후 저장하기 버튼을 클릭하면 변경 사항이 즉시 반영됩니다.
