학부모 대시보드

로그인 후 자녀의 학교 생활 정보를 한눈에 확인할 수 있는 메인 화면입니다.

자녀 등하교 상태

대시보드 최상단에 자녀의 등하교 상태가 표시됩니다.

  • 등교 완료 시 통과 시간이 함께 표시됩니다.
  • 하교 후에는 하교 상태로 변경됩니다.

상단 위젯

  • 자녀 다음 수업 - 자녀의 현재 시간 기준 다음 수업 정보가 표시됩니다.
  • 오늘의 급식 - 학교 급식 메뉴를 확인합니다.
  • 상담 신청 - 클릭하면 담임선생님께 상담을 신청할 수 있는 모달이 열립니다.

자녀 시간표

오늘 요일 기준 자녀의 교시별 시간표가 표시됩니다.

자녀의 시간표가 표시되지 않으면, 자녀 학생 계정에서 시간표가 설정되어 있는지 확인하세요.

공지사항

학교 전체 공지사항이 최신순으로 표시됩니다. 클릭하면 공지사항 상세 페이지로 이동합니다.

메시지

최근 받은 메시지 목록이 표시됩니다.

  • 읽지 않은 메시지가 있으면 배지로 수량이 표시됩니다.
  • 담임선생님과 1:1 메시지를 주고받을 수 있습니다.

일정 (캘린더)

월별 캘린더에서 학교 일정과 개인 일정을 관리합니다.

  • 날짜를 클릭하면 해당 날짜의 일정이 표시됩니다.
  • + 버튼으로 새 일정을 등록할 수 있습니다.

사이드바 메뉴

메뉴설명
대시보드현재 페이지 (메인 화면)
학급정보자녀의 학급 공지, 과제, 상담, 출결, 통신문 확인
입시상담AI 기반 대학 입시 분석 및 상담

포인트

사이드바에 보유 포인트가 표시됩니다. 포인트 충전 버튼으로 충전 화면에 진입할 수 있습니다.
